# Barumi

### 시각 장애인을 위한 대중교통 도움 지팡이 및 버스 정보 앱 개발

![image](https://github.com/user-attachments/assets/063c3acf-bbf8-40dd-8215-67ce2cdc7882)

## 스마트 지팡이 및 모바일 어플리케이션
이 프로젝트는 시각 장애인을 위한 스마트 지팡이와 이를 지원하는 모바일 어플리케이션을 개발하는 것을 목표로 합니다. 지팡이는 조도 센서와 라이다 센서를 이용하여 안전하고 편리한 이동을 지원하며, 어플리케이션은 실시간 대중교통 정보를 제공합니다.

**스마트 지팡이 기능**
1. 조도 센서
- 설정한 밝기 값 이하로 조도가 떨어지면 LED가 점등됩니다.
- 설정한 밝기 값 이상으로 조도가 올라가면 LED가 소등됩니다.

2. 라이다 센서
- 장애물을 감지하여 사용자의 안전을 보장합니다.
- 블루투스 데이터 전송

장애물 감지 데이터를 블루투스를 통해 모바일 어플리케이션으로 전송합니다.

**모바일 어플리케이션 기능**
1. 공공 데이터 API 활용
- 서울 버스 위치 정보 조회
- 서울 버스 도착 정보 조회
- 대중교통 환승 경로 데이터 수신
- 버스 정보 확인

실시간으로 현재 버스 정보를 확인할 수 있습니다.

경유 버스 정보 제공
- 사용자가 경유하는 버스에 대한 정보 제공
